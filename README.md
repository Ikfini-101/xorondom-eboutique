# Xorondom · E‑Boutique — Starter

EL10 • sans argent • Node/TS/Express + Jest + Playwright.

## Démarrage
```bash
npm install
npm run dev
```
Tests:
```bash
npm run test:unit
npm run test:e2e
```

## Structure
- `src/` code applicatif (Express, services, utils)
- `tests/unit/` tests unitaires (Jest)
- `tests/e2e/` tests E2E (Playwright)
- `seeds/` données de démonstration
- `.windsurf/rules/` règles persistantes Windsurf (contexte)
